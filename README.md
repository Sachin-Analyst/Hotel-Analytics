# Hotel-Analytics
Welcome to the Hotel Analytics project. This repository features a data-driven analysis of the AtliQ Hotel Group's performance across four major Indian cities. The project utilizes quantitative modeling to evaluate revenue, occupancy, and guest satisfaction to optimize marketing spend and operational efficiency

## Project Description 
This project addresses inconsistent revenue generation across luxury and business property segments. By analyzing internal booking logs and customer feedback, this study identifies high-performing properties and evaluates the effectiveness of various booking platforms like MakeYourTrip and Logtrip
